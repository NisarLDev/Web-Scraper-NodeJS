# Web-Scrapper-NodeJS

### NodeJS Version used

18.16

### Dependencies
 Express.
 
 Cheerios.
 
 Axios.

 Nodemon.

To install dependencies:

1 - npm i express

2 - npm i cheerio

3 - npm i axios

4 - sudo npm i nodemon -g
